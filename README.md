# SalesDock

SalesDock is an interactive data analytics dashboard built using Streamlit, Plotly, and Pandas. It visualizes Superstore sales data with dynamic filters and interactive charts, enabling users to explore trends, compare regional performance, analyze product categories, and generate meaningful insights for decision-making.

## Features

- Upload support for **CSV**, **TXT**, **XLSX**, and **XLS** files
- Automatically uses a default sample dataset if none is uploaded
- Date filtering with start and end selection
- Sidebar filters for **Region**, **State**, and **City**
- Multiple visualization types including:
  - Bar charts
  - Pie charts
  - Time series line charts
  - Treemap hierarchy views
  - Scatter plots
- Download options for filtered and summary data
- Expandable styled tables and pivot views
